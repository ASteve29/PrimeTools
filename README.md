# Prime-Counter

Fast prime number calculations using C++ and Cython for Python.

---

## Description

Prime-Counter provides high-performance prime number utilities. It uses a C++ backend for speed and a Cython wrapper for Python usability. You can check if a number is prime, generate lists of primes, or use it as a starting point for math projects.

---

## Features

- Fast prime checking
- Prime number generation
- C++ backend for performance
- Cython wrapper for easy Python use

---

## Requirements

- Python 3.12+
- C++ compiler (g++, clang++, or MSVC)
- Cython
